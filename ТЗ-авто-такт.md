# ТЗ: Автоматизация тактического сражения

## Контекст

На локациях тактического сражения (100 = большой такт, 101 = малый, 102 = аниме) нужна полная автоматизация движения по базам. Кнопка управления — плавающая через `createDisableButton()`.

**Обработка боя не требует отдельного кода.** Стандартный fight-процессинг уже поддерживает всё необходимое:
- `wor_fight_only_mobs = false` → бот будет работать с реальными игроками (не скипать их)
- `wor_fight_give_up_active = true` → авто-сдача
- Проверка травмы, скип-лист, уровень — всё из настроек автоматически

Пользователь просто настраивает нужные флаги перед запуском авто-такта.

## Шаги реализации

### Шаг 1. Создать `Territory/TaktAuto.js`

Файл определяет глобальную функцию `handleAutoTakt(baseNames, container, myTeam, t, delay)`.

**Структура файла:**

```javascript
async function handleAutoTakt(baseNames, container, myTeam, t, delay) {
    // 1. Парсим состояние баз ИЗ ОРИГИНАЛЬНОГО HTML (до того как processTaktCommon его изменит)
    const bases = parseTaktBases(baseNames, container);

    // 2. Создаём кнопку управления — только если включена в popup
    const taktActive = await CommonHelper.getExtStorage('wor_takt_active');
    const showButton = await CommonHelper.getExtStorage('wor_takt_opt_active_button');

    if (showButton) {
        CommonHelper.createDisableButton(
            taktActive ? 'Остановить авто-такт' : 'Включить авто-такт',
            async () => {
                await CommonHelper.setExtStorage('wor_takt_active', !taktActive);
                CommonHelper.reloadPage();
            }
        );
    }

    // 3. Запускаем автодвижение если активно
    if (!taktActive) return;

    await runAutoTakt(t, bases, myTeam, delay);
}

function parseTaktBases(baseNames, container) {
    const bases = [];
    const html = container.innerHTML;
    for (const [pointId, name] of Object.entries(baseNames)) {
        const regex = new RegExp(name + ':<span[^>]*>№(\\d+)<\\/span>');
        const match = html.match(regex);
        bases.push({
            id: parseInt(pointId),
            name,
            team: match ? parseInt(match[1]) : null  // null = нейтральная
        });
    }
    return bases;
}

async function runAutoTakt(t, bases, myTeam, delay) {
    // Цели: базы не под нашим контролем (нейтральные или вражеские)
    const targets = bases.filter(b => b.team !== myTeam);

    if (targets.length === 0) {
        // Все базы наши — ждём и перезагружаемся
        CommonHelper.log('Авто-такт: все базы захвачены, ждём...');
        await CommonHelper.delay(10000, 20000);
        CommonHelper.reloadPage();
        return;
    }

    // Случайный выбор цели
    const target = targets[Math.floor(Math.random() * targets.length)];
    CommonHelper.log('Авто-такт: идём к базе ' + target.name);

    const renderGrid = (json) => CommonHelper.renderMiniGridInto(
        document.getElementById('gridA'), json.grid || []
    );

    await t.toPoint(target.id, delay, renderGrid);
    // toPoint навигирует через json.realUrl → страница перезагружается → цикл продолжается
}
```

### Шаг 2. Изменить `Territory/process-territory.js`

В функции `processTaktCommon` добавить **один вызов** до начала цикла `innerHTML.replace()`:

```javascript
async function processTaktCommon(baseNames, delay) {
    const t = new Territory();
    const container = document.querySelector('.contur');
    if (!container) return;

    // Извлекаем номер нашей команды
    const teamMatch = container.textContent.match(/Вы в команде №(\d+)/);
    const myTeam = teamMatch ? parseInt(teamMatch[1], 10) : null;

    // ↓ ДОБАВИТЬ ЗДЕСЬ — до innerHTML.replace()
    await handleAutoTakt(baseNames, container, myTeam, t, delay);
    // ↑ конец добавленного кода

    // дальше идёт существующий код — innerHTML.replace(), клик-листенеры и т.д.
    for (const [pointId, name] of Object.entries(baseNames)) {
        container.innerHTML = container.innerHTML.replace(
            ...
        );
    }
    // ...
}
```

> **Почему до replace()**: `handleAutoTakt` парсит HTML до модификации DOM, иначе имена баз будут уже заменены на `<span>`.

## Edge cases

- Все базы захвачены моей командой → ждём 10–20 сек, перезагружаемся
- Нейтральные базы (team === null) → включаются в цели наравне с вражескими
- Бой во время движения → стандартный процессинг боя (fight settings пользователя)
- После сдачи в бою → страница возвращается на тактическую карту → цикл продолжается

- `wor_fight_only_mobs = false` — разрешить бой с игроками
- `wor_fight_give_up_active = true` — авто-сдача

## Проверка

1. Открыть тактическую карту (location 100/101/102)
2. Кнопка "Включить авто-такт" появляется внизу справа (если включена "Кнопка запуска" в popup)
3. Нажать → `wor_takt_active = true` → страница перезагружается → кнопка "Остановить авто-такт"
4. То же самое что переключить чекбокс "Автотакт" в popup
5. Персонаж начинает идти к случайной базе не под контролем команды
6. При встрече с игроком → стандартный бой → авто-сдача → возврат на такт-карту → продолжение
