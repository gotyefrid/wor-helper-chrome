# WOR Helper

Chrome-расширение для автоматизации браузерной игры **World of Rest** (WOR).

---

## Установка

### Из Chrome Web Store

Расширение доступно в [Chrome Web Store](https://chrome.google.com/webstore/). Удобный вариант, но обновления там выходят реже, чем в этом репозитории.

### Вручную (актуальная версия)

1. Скачайте или клонируйте репозиторий на компьютер.
2. Откройте `chrome://extensions/` в браузере.
3. Включите **Режим разработчика** (правый верхний угол).
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите папку с проектом.

> После обновления файлов расширения нажмите кнопку обновления (↺) рядом с расширением на странице `chrome://extensions/`.

## Возможности

| Модуль | Описание |
|--------|----------|
| **Рыбалка** | Автоматически закидывает удочку, ждёт поклёвку, подсекает |
| **Алхимия** | Автоматический сбор трав / алхимическое крафтение |
| **Рудокоп** | Покупает пропуск на шахту, запускает добычу, собирает лут |
| **Сражение** | Автобой с настройкой типа удара, зельями HP/MA, списком исключений |
| **Разбойники** | Отдельный режим для боёв с разбойниками |
| **Авто-такт** | Автоматическое движение и бой в режиме тактической карты |
| **История карты** | Запись и воспроизведение маршрутов, обход всех точек |
| **Капча** | Передаёт капчу на внешний HTTP-сервис и вводит ответ |
| **Чат** | Парсинг сообщений, пересылка в Telegram, быстрые адресаты, автоответ |
| **Квест Чернокнижника** | Полный авторан: принять → пройти лабиринт → получить награду |
| **Уведомления Telegram** | Отправка событий и ошибок в Telegram-бот |
| **Движение клавишами** | Навигация по карте стрелками / WASD |
| **Авто-логин** | Автоматическая авторизация при открытии страницы входа |
