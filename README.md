# WOR Extension

Chrome-расширение для автоматизации браузерной игры **World of Rest** (WOR).

## Возможности

| Модуль | Описание |
|--------|----------|
| **Рыбалка** | Автоматически закидывает удочку, ждёт поклёвку, подсекает, переходит к следующей точке |
| **Рудокоп** | Покупает пропуск на шахту, запускает добычу, собирает лут, повторяет цикл |
| **Алхимия** | Автоматический сбор трав / алхимическое крафтение |
| **Сражение** | Автобой с настройкой типа удара, зельями HP/MA, списком исключений по имени и уровню |
| **Разбойники** | Отдельный режим обработки случайных атак разбойников |
| **Авто-такт** | Автоматическое движение и бой в режиме тактической карты |
| **История карты** | Запись и воспроизведение маршрутов по карте, обход всех точек |
| **Капча** | Передаёт изображение капчи на внешний HTTP-сервис и вводит ответ |
| **Чат** | Парсинг входящих сообщений, быстрые адресаты, автоответ по ключевым словам |
| **Квест Чернокнижника** | Полный авторан квеста: принять → пройти лабиринт → завершить → получить награду |
| **Парсинг страниц** | Периодический обход заданных URL с поиском целевых строк |
| **Авто-логин** | Автоматическая авторизация при открытии страницы входа |
| **Уведомления Telegram** | Отправка событий и ошибок в Telegram-бот |
| **Движение клавишами** | Навигация по карте стрелками/WASD |

---

## Установка

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика** (правый верхний угол)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку с проектом

> После любых изменений в `manifest.json` необходимо перезагрузить расширение на этой же странице.

## Настройка

Откройте popup расширения (клик по иконке в тулбаре Chrome). Все настройки сохраняются в `chrome.storage.local`.

### Персонаж

| Поле | Описание |
|------|----------|
| Ник | Имя персонажа (используется в некоторых запросах) |
| Пароль | Для авто-логина при открытии страницы входа |

### Рыбалка

| Настройка | Описание |
|-----------|----------|
| Включить | Запускает автоматическую рыбалку |
| Кнопка запуска | Показывает кнопку прямо на странице игры |
| Приоритет | Выбор целевого вида рыбы (карась, окунь, карп, … белуга) |

### Алхимия

| Настройка | Описание |
|-----------|----------|
| Включить | Автоматический сбор/крафт |
| Кнопка запуска | Показывает кнопку на странице |

### Рудокоп

| Настройка | Описание |
|-----------|----------|
| Включить | Запускает авто-добычу |
| Кнопка запуска | Показывает кнопку на странице |

### Сражение

| Настройка | Описание |
|-----------|----------|
| Включить | Запускает автобой |
| Кнопка запуска | Показывает кнопку на странице |
| Кого НЕ бить | Список имён / подстрок (напр. `Серебряный дракон [25]`); бот пропускает таких врагов |
| Сдаваться | Сдаваться вместо того чтобы драться |
| Тип удара | Физический (1) или Магический (2) |
| Пить HP | Автоматически пить зелье здоровья |
| HP% для зелья | Порог HP (%) при котором пить зелье (например `70` = пить при HP < 70%) |
| Пить MA | Автоматически пить зелье маны |
| MA% для зелья | Порог MA (%) |
| Не бить лвл >= | Пропускать врагов с уровнем выше или равным указанному |
| Проверять травму | Останавливать бот при наличии травмы |
| Травма >= часов | Минимальное кол-во часов травмы для остановки |
| Активировать приманку | Нажимать кнопку приманки перед боем |
| Только мобы | Биться только с мобами (не с игроками) |
| Мин. урон | Если урон по врагу ниже порога — бот останавливается и пишет в Telegram |

### Авто-такт

| Настройка | Описание |
|-----------|----------|
| Включить | Автоматическое движение и бой на тактической карте |
| Кнопка запуска | Показывает кнопку на странице |
| Ходить рандомно | Выбирать случайную клетку для хода |

### Разбойники

Включить/выключить авто-бой с разбойниками. Настройки HP/MA-зелий берутся из раздела **Сражение**.

### История карты

| Настройка | Описание |
|-----------|----------|
| Включить | Запись и воспроизведение маршрутов |
| Сброс (↻) | Очистить историю перемещений |
| Задержка "от,до", мс | Задержка между шагами. Одно число (`50`) или диапазон (`10,50`) |
| Обход всех точек | Последовательно посещать все сохранённые точки маршрута |
| Перезагружать "от,до", сек | Перезагружать страницу каждые N секунд (0 = выключено) |

### Проходить капчу

| Настройка | Описание |
|-----------|----------|
| Включить | Автоматически решать капчу |
| Хост | URL внешнего сервиса распознавания (напр. `http://localhost:5000`) |

Сервис должен принимать POST-запрос на `/detect_puzzle` с полем `file` (изображение капчи) и возвращать JSON с результатом.

### Уведомления (Telegram)

| Настройка | Описание |
|-----------|----------|
| Включить | Отправлять уведомления в Telegram |
| Бот уведомлений API | Token бота для общих уведомлений (`wor_tg_bot_common_token`) |
| Бот чата API | Token бота для сообщений из чата (`wor_tg_bot_chat_token`) |
| Chat ID | ID чата/пользователя для отправки (`wor_tg_chat_id`) |

### Чат

| Настройка | Описание |
|-----------|----------|
| Парсить сообщения | Читать входящие сообщения и пересылать в Telegram |
| Интервал "от,до", сек | Случайный интервал опроса чата. Одно число (`10`) или диапазон (`5,30`). По умолчанию `5,30` |
| Быстрые адресаты | Список кнопок для быстрой отправки; формат `кнопка:кому` или `кнопка:кому:1` (приватное) |
| Автоответ | Правила автоответа по ключевым словам; формат `слово:ответ1\|ответ2` |

### Квесты

| Настройка | Описание |
|-----------|----------|
| Чернокнижник | Автоматически принимать, проходить и завершать квест чернокнижника |

### Парсинг

Периодически открывает заданные ссылки и ищет на странице целевые строки. Удобно для мониторинга событий в игре.

| Настройка | Описание |
|-----------|----------|
| Включить | Активировать парсинг |
| Ссылки и цели | Список URL и строк-целей для поиска |
| Таймаут, сек | Интервал между проверками |
| Цели должны отсутствовать | Инвертировать поиск (срабатывать когда цель НЕ найдена) |

### Лог

Включить вывод отладочных сообщений в консоль DevTools.

### Service Worker

`Backend/background.js` регистрирует два alarm-а:

| Alarm | Интервал | Действие |
|-------|----------|----------|
| `sendMessagesAlarm` | 5–33 секунды (рандом) | Отправляет накопленные сообщения чата |
| `disableChaosBattle` | 30 минут | Отключает хаос-бой |

`sendMessagesAlarm` — одноразовый alarm: после каждого срабатывания планируется следующий с новым случайным интервалом. Диапазон настраивается в popup (поле **Интервал "от,до", сек** в разделе Чат), по умолчанию `5,30`. Значения fallback хранятся в константах `CHAT_ALARM_MS_FROM_DEFAULT` / `CHAT_ALARM_MS_TO_DEFAULT` в `Backend/background.js`.

Также слушает `chrome.tabs.onUpdated` и восстанавливает вкладку игры при краше (`chrome-error://`).

### Content Scripts — порядок загрузки

На **всех** страницах `/wap/*` загружаются (в этом порядке):

1. `Common/moving-key.js`
2. `Common/CommonHelper.js`
3. `Chat/luxon.js`
4. `Chat/Chat.js`
5. `Common/process-common.js`
6. `Common/process-commands.js`
7. `Quest/WarlockQuestRunner.js`
8. `Quest/process-warlock-runner.js`

Остальные модули подключаются только на соответствующих URL (см. `manifest.json`).

### Задержки и случайность

Все действия имеют случайный компонент для имитации поведения человека:

```javascript
CommonHelper.EXTRA_SMALL_RANDOM  // 100–300 мс
CommonHelper.SMALL_RANDOM        // 300–700 мс
CommonHelper.MEDIUM_RANDOM       // 1000–3000 мс
CommonHelper.LARGE_RANDOM        // 3000–5000 мс
CommonHelper.EXTRA_LARGE_RANDOM  // 5000–7000 мс
CommonHelper.XXL_RANDOM          // 7000–15000 мс

// Произвольный диапазон:
await CommonHelper.delay(500, 1500);
```

## Отладка

| Контекст | Как открыть |
|----------|-------------|
| Content script | DevTools на странице игры → Console |
| Service Worker | `chrome://extensions/` → "Инспектировать представления: service worker" |
