# История карты

Отслеживает посещённые клетки карты и помогает автоматически перемещаться по территории. Раздел сворачивается/разворачивается кликом на заголовок.

| Настройка | Тип | Описание |
|-----------|-----|----------|
| **История карты** (переключатель) | Вкл/Выкл | Включает отслеживание и подсветку посещённых клеток на карте. |
| **↻ (сброс)** | Кнопка | Сбрасывает всю историю посещений — все клетки станут непосещёнными. |
| **Задержка "от,до", мс** | Текст | Задержка между шагами при авто-перемещениях. Можно указать одно число (`100`) или диапазон (`50,150`). По умолчанию: `50,100`. |
| **Обход всех точек** | Вкл/Выкл | Включает режим автоматического обхода всех непосещённых клеток текущей локации. |
| **Перезагружать "от,до", сек** | Текст | Автоматически перезагружать страницу каждые N секунд. Можно указать диапазон (`300,600`). `0` = выключено. |

---

## Подсветка клеток на карте

Когда история карты включена, клетки на карте подсвечиваются:
- **Белая точка** — клетка уже посещалась.
- **Красная точка** — клетка ещё не посещалась.

История хранится отдельно для каждой локации.

---

## Кнопки быстрого перемещения

На странице территории расширение добавляет в меню кнопки быстрого перехода к ключевым точкам локации (шахта, город, подземелье и т.д.). Набор кнопок зависит от текущей локации.

---

## Обход всех точек

При включении режима "Обход всех точек" расширение автоматически строит маршрут по всем непосещённым клеткам текущей локации и последовательно их обходит. Когда все клетки посещены — выводится сообщение и режим выключается.

Остановить обход можно:
- Кнопкой "Остановить обход всех точек" в меню на странице.
- Клавишей `M` (если включено "Движение клавишами").
- Сняв чекбокс "Обход всех точек" в настройках расширения (popup).

---

## Автоперезагрузка страницы

Если задан параметр "Перезагружать от,до" — страница будет автоматически перезагружаться через случайный интервал в указанном диапазоне. Используется чтобы стоять на одном месте на природе и бить мобов: после каждой перезагрузки на персонажа снова могут нападать.
