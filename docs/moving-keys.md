# Движение клавишами

Позволяет управлять персонажем на карте с клавиатуры, а также выполнять действия в бою и в квесте лабиринта Чернокнижника.

| Настройка | Тип | Описание |
|-----------|-----|----------|
| **Движение клавишами** (переключатель) | Вкл/Выкл | Включает/выключает управление с клавиатуры. |

> Управление не работает, если фокус находится в текстовом поле или textarea.

**Клавиши передвижения по карте:**

| Клавиша | Действие |
|---------|----------|
| `W` | Вверх |
| `S` | Вниз |
| `A` | Влево |
| `D` | Вправо |
| `Q` | Вверх-влево (по диагонали) |
| `E` | Вверх-вправо (по диагонали) |
| `Z` | Вниз-влево (по диагонали) |
| `C` | Вниз-вправо (по диагонали) |

**Клавиши действий:**

| Клавиша | Действие |
|---------|----------|
| `Пробел` | Нанести удар в бою (тип удара — выбранный в настройках). Если форма боя не найдена — переход на `teritory.php`. |
| `F` | Нанести физический удар в бою. Если форма боя не найдена — переход на `teritory.php`. |
| `R` | Сдаться в бою (ссылка `killme`). Если ссылка не найдена — переход на `teritory.php`. |
| `M` | Остановить авто-обход карты (История карты → Обход всех точек). Если остановить не удалось — переход на `teritory.php`. |
