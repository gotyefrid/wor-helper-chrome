# Чат

Функции для работы с игровым чатом. Раздел сворачивается/разворачивается кликом на заголовок.

| Настройка | Тип | Описание |
|-----------|-----|----------|
| **Парсить сообщения** | Вкл/Выкл | Включает периодическое чтение чата и пересылку сообщений в Telegram. Требуется для работы автоответа. |
| **Интервал "от,до", сек** | Текст | Как часто читать чат. Можно указать диапазон (`5,30`). По умолчанию: `5,30`. |
| **Быстрые адресаты** | Ссылка | Открывает настройку быстрых кнопок чата (см. ниже). |
| **Автоответ** | Ссылка | Открывает настройку автоответа на личные сообщения (см. ниже). |

---

## Пересылка сообщений в Telegram

Когда включено "Парсить сообщения", сервис-воркер периодически читает чат и отправляет новые сообщения в Telegram. Интервал случайный — берётся из настройки "Интервал от,до" (по умолчанию `5,30` секунд):

- Все сообщения пересылаются в **бот чата** (полный архив).
- Личные сообщения и упоминания вашего ника дополнительно отправляются в **бот уведомлений** с уведомлением 🔔.
- Системные сообщения о травме, штрафах, новых письмах, изъятии/получении предметов также дублируются в бот уведомлений.

---

## Быстрые адресаты

Добавляет кнопки быстрого заполнения формы чата прямо на странице. Кнопки появляются над строкой ввода.

Настройки открываются в модальном окне:

| Поле | Формат | Описание |
|------|--------|----------|
| **Включено** | Вкл/Выкл | Показывать/скрывать быстрые кнопки на странице чата. |
| **Адресаты** | `Название:Ник` или `Название:Ник:1` | Кнопки для быстрой подстановки адресата. Третий параметр `:1` автоматически ставит галочку "приватное сообщение". Пример: `Админу:Misha:1` |
| **Сообщения (Теги)** | `Название:текст` | Кнопки для быстрой вставки текста в поле сообщения. Пример: `Ссылка на бой:[log][/log]` |

---

## Автоответ

Автоматически отвечает на личные сообщения в чате.

> Требует включённого "Парсить сообщения".

Настройки открываются в модальном окне:

| Поле | Описание |
|------|----------|
| **Включено** | Включает/выключает автоответ. |
| **Пауза, минут** | Минимальный интервал между автоответами. Не будет отвечать чаще указанного значения. По умолчанию: 5 минут. |
| **От кого отвечать** | Кому отвечать: `Модераторы` (список модераторов клана), `Все`, или конкретные ники. Список модераторов кэшируется на 48 часов. |
| **Правила ответов** | Формат: `ключевое_слово:ответ1\|ответ2\|ответ3`. Если во входящем сообщении найдено ключевое слово — выбирается случайный ответ из списка. Если ни одно правило не подошло — отправляется случайный смайл из набора по умолчанию. |

## Контекстное меню в чате

На странице чата при клике правой кнопкой мыши по нику игрока появляется меню:

| Пункт | Действие |
|-------|----------|
| **Перейти в профиль** | Открывает страницу профиля игрока в новой вкладке. |
| **Отправить WR** | Открывает модальное окно для передачи WR игроку прямо со страницы чата. |
| **Отправить на почту** | Открывает модальное окно для отправки письма игроку прямо со страницы чата. |
| **Копировать ник** | Копирует ник игрока в буфер обмена. |

> Чтобы вызвать стандартное контекстное меню браузера вместо этого — удерживайте `Shift` при клике правой кнопкой.

## Предпросмотр боя

При наведении на ссылку боя (вида `Бой №123456`) в чате появляется всплывающее окно с таблицей участников и результатами боя.
