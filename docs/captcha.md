# Проходить капчу

Автоматически решает капчу-пазл, которая появляется в игре. Для работы требуется внешний сервис распознавания.

Раздел сворачивается/разворачивается кликом на заголовок.

| Настройка | Тип | Описание |
|-----------|-----|----------|
| **Проходить капчу** (переключатель) | Вкл/Выкл | Включает/выключает авторешение капчи. |
| **Хост** | Текст | URL внешнего сервиса распознавания капчи. Должен начинаться с `http`. |

**Как работает:**
1. При появлении страницы капчи расширение делает скриншот изображения пазла.
2. Отправляет его на внешний сервис.
3. Получает координаты куда поставить пазл.
4. Устанавливает пазл на нужное место и отправляет форму.

**Защита от ошибок:**
- Если пазл поставлен неверно — картинка обновляется и делается повторная попытка.
- После 3 неудачных попыток подряд — отправляется срочное уведомление в Telegram и решение останавливается.
- Капча не решается повторно, если с момента последнего решения прошло менее 7 минут.
- Если изменилась структура страницы или хэш ресурсов игры — решение останавливается и отправляется уведомление в Telegram (защита от неожиданных обновлений игры).

**Если капча выключена:**
- При появлении капчи в Telegram отправляется уведомление со ссылкой на страницу.
- Через 15 секунд происходит перезагрузка или возврат к предыдущей странице.

## API внешнего сервиса

Расширение обращается к хосту по следующему эндпоинту:

**`POST {хост}/detect_puzzle`**

**Тело запроса:** `multipart/form-data`

| Поле | Тип | Описание |
|------|-----|----------|
| `file` | файл (PNG) | Изображение капчи, снятое со страницы игры. |

**Ответ:** JSON-объект с координатами куда нужно поставить пазл:

```json
{
  "x": 123,
  "y": 45
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| `x` | number | Координата X (в пикселях) левого края пазла. |
| `y` | number | Координата Y (в пикселях) верхнего края пазла. |
