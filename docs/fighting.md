# Сражение

Автоматизирует бои с противниками. Раздел сворачивается/разворачивается кликом на заголовок.

| Настройка | Тип | Описание |
|-----------|-----|----------|
| **Сражение** (переключатель) | Вкл/Выкл | Включает/выключает автобой. |
| **Кнопка запуска** | Вкл/Выкл | Показывает/скрывает кнопку включения/выключения боёв прямо на странице игры. |
| **Кого НЕ бить** | Список | Список противников которых бот не будет бить. Можно указать часть имени (например `дракон` — пропустит всех драконов) или полное имя с уровнем (`Красный дракон [25]`). При встрече с таким противником — отправляется уведомление в Telegram и страница перезагружается. |
| **Сдаваться** | Вкл/Выкл | Если включено — бот сразу сдаётся в любом бою вместо того чтобы атаковать. |
| **Тип удара** | Список | Тип атаки: Физический или Магический. |
| **Пить HP** | Вкл/Выкл | Автоматически пить зелья здоровья в бою. |
| **HP% для зелья** | Число | При каком проценте HP начинать пить зелья здоровья. Например: `70` = пить когда HP ниже 70%. |
| **Пить MA** | Вкл/Выкл | Автоматически пить зелья маны в бою. |
| **MA% для зелья** | Число | При каком проценте маны начинать пить зелья. |
| **Не бить лвл >=** | Число | Пропускать противников указанного уровня и выше. Бот не атакует и ждёт окончания боя. |
| **Проверять травму** | Вкл/Выкл | Если включено — бот проверяет текущее время травмы перед атакой. |
| **Травма >= часов** | Число | Если накопленная травма превышает указанное количество часов — бот не атакует. |
| **Активировать приманку** | Вкл/Выкл | Автоматически активировать приманку когда она заканчивается. |
| **Только мобы** | Вкл/Выкл | Если включено — бот не атакует живых игроков, только монстров. При нападении игрока — отправляется уведомление в Telegram. |
| **Мин. урон** | Число | Если бот наносит урон ниже этого значения — останавливается и отправляет уведомление в Telegram. `0` = проверка отключена. Добивающие удары (когда у врага 0 HP) не учитываются. |
| **Выходить на природу** | Вкл/Выкл | После окончания боя переходить напрямую на страницу территории (Природа), а не нажимать кнопку "Выход из боя" (которая может вести в другое место). |

---

## Как работает автобой

1. При попадании на страницу боя — определяет противника и проверяет все условия (стоп-лист, уровень, травма, тип противника).
2. Если все проверки пройдены — пьёт зелья при необходимости и наносит удар.
3. На странице ожидания (между ходами) — перезагружает страницу.
4. После окончания боя — возвращается по сохранённой ссылке возврата (страница ожидания рыбалки, сбора трав, добычи руды и т.д., чтобы не потерять прогресс профессии), либо выходит на природу если ссылки возврата нет.

## Защита от аномалий

- **Мало урона**: если урон по врагу ниже порога из настройки "Мин. урон" — бот останавливается и пишет в Telegram.
- **Серия уворотов**: если противник уворачивается более 5 раз подряд — бот останавливается и пишет в Telegram.
- **Вмешательство игрока**: если игрок сам вошёл в бой вручную (через "Вмешаться") — бот не атакует и ждёт 5 секунд. Проверка работает так: при каждом ходе расширение загружает страницу чата и ищет системные сообщения. Если найдено "Вы вмешались в бой!" и ещё нет "Бой №X закончен" — бот отступает. Результат кэшируется на 20 секунд — то есть в затяжном бою чат запрашивается примерно раз в 20 секунд, а не при каждом ходе.

## Зелья

Зелья пьются в порядке от меньшего к большему: сначала используются банки на 10, затем 20, 50, 100, 250, 500. Зелья должны быть в инвентаре персонажа на странице боя.
