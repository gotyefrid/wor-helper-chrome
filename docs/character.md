# Персонаж

| Поле | Описание |
|------|----------|
| **Ник** | Ник персонажа в игре. Используется в функциях чата для идентификации своих сообщений. |
| **Пароль** | Пароль от аккаунта. Нужен **только** для авто-авторизации — когда сессия истекает, расширение само вводит логин и пароль на странице входа. Больше нигде не используется. |

> Пароль хранится локально в `chrome.storage.local` и никуда не передаётся, кроме самой страницы авторизации игры.
