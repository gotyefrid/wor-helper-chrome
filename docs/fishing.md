# Рыбалка

Включает автоматическую рыбалку. Раздел сворачивается/разворачивается кликом на заголовок.

| Настройка | Тип | Описание |
|-----------|-----|----------|
| **Рыбалка** (переключатель) | Вкл/Выкл | Включает/выключает авторыбалку. Когда активна — расширение автоматически забрасывает удочку, ждёт поклёвки, подсекает и повторяет цикл. |
| **Кнопка запуска** | Вкл/Выкл | Показывает/скрывает кнопку запуска рыбалки прямо на странице игры. Удобно для быстрого старта без открытия popup. При отключении рыбалки через эту кнопку также отключаются бои. |
| **Приоритет** | Список | Целевая рыба. Если поймана именно она — расширение возвращается на ту же локацию. Если поймана другая — выбирается случайная локация. |

**Требования для запуска:**
- На персонаже надета удочка.
- Персонаж находится на территории, где доступна рыбалка (есть ссылка "Ловить рыбу").

> Если удочка не надета — рыбалка и бой автоматически выключаются, в Telegram отправляется уведомление.

**Как работает авторыбалка:**
1. На странице территории находит ссылку "Ловить рыбу" и переходит по ней.
2. Выбирает локацию — предпочитает ту, где поймали нужную рыбу (по настройке "Приоритет"), иначе случайную.
3. На странице ожидания ждёт поклёвки и кликает "Подсечь". Рядом со стандартным таймером игры отображается счётчик в формате "X из Y секунд".
4. Если во время ожидания поклёвки на персонажа нападают и появляется кнопка "В бой" — страница автоматически перезагружается.
5. Цикл повторяется.
