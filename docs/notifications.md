# Уведомления

Настройка отправки уведомлений в Telegram. Раздел сворачивается/разворачивается кликом на заголовок.

| Настройка | Тип | Описание |
|-----------|-----|----------|
| **Уведомления** (переключатель) | Вкл/Выкл | Включает/выключает отправку всех уведомлений в Telegram. Если выключено — никакие сообщения не отправляются, даже при ошибках. |
| **Бот уведомлений API** | Текст | API-токен Telegram-бота для системных уведомлений (ошибки, события, статусы). Получить можно у @BotFather в Telegram. |
| **Бот чата API** | Текст | API-токен отдельного Telegram-бота для пересылки сообщений из игрового чата. |
| **Chat ID** | Текст | ID чата или пользователя в Telegram, куда отправляются сообщения. Один ID используется для обоих ботов. |

---

## Два бота

Расширение использует два отдельных бота:

- **Бот уведомлений** — отправляет системные события: ошибки, предупреждения, статусы (нет удочки, мало урона, капча и т.д.).
- **Бот чата** — пересылает сообщения из игрового чата в Telegram (используется в разделе "Чат").

Можно использовать один и тот же токен для обоих ботов.

---

## Защита от дублей

Для бота уведомлений действует защита от повторной отправки одинаковых сообщений: если то же самое сообщение уже отправлялось недавно — повторная отправка пропускается. Время жизни дедупликации задаётся каждым конкретным событием (обычно от 1 до 20 минут).
