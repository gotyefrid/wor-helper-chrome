# Алхимия

Автоматизирует сбор трав на территории. Раздел сворачивается/разворачивается кликом на заголовок.

| Настройка | Тип | Описание |
|-----------|-----|----------|
| **Алхимия** (переключатель) | Вкл/Выкл | Включает/выключает авто-сбор трав. Одновременно включает/выключает бои. |
| **Кнопка запуска** | Вкл/Выкл | Показывает/скрывает кнопку запуска прямо на странице игры. При отключении через эту кнопку также отключаются бои. |

**Требования для запуска:**
- Персонаж находится на странице территории (Природа), где доступна кнопка "Осмотреться".

**Как работает:**
1. На странице территории находит кнопку "Осмотреться" и нажимает её — запускает поиск трав.
2. На странице ожидания ждёт появления кнопки "Собрать".
3. Когда трава найдена — нажимает "Собрать".
4. Цикл повторяется.

**Особые ситуации:**
- Если во время ожидания сбора на персонажа нападают и появляется кнопка "В бой" — страница перезагружается.
- Если кнопка "Осмотреться" не найдена — в Telegram отправляется уведомление, скрипт ничего не делает.
- Если не найдена ссылка "Природа" для возврата — алхимия и бои автоматически выключаются, в Telegram отправляется уведомление.
